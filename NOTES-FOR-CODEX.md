# Notes For Codex

This repository is a download hub for 3D printer upgrades, conversions, macros, and related community project files.

## Primary goal

Keep the repo easy for other people to browse, understand, and download from.

## Preferred direction

- Organize projects clearly by category and printer
- Keep each project in its own folder
- Include a `README.md` for each project whenever possible
- Make the repo friendly for beginners and people downloading files directly
- Prefer clean naming and simple structure over deep nesting

## Current categories

- `Conversions/`
- `macros/`

## Good future improvements

- Add preview images or photos for projects
- Add better per-project install and hardware notes
- Reduce confusing nested folders where possible
- Keep old versions separated from the newest recommended files
- Add more printer-specific project pages as the repo grows

## Working style

The repo owner is newer to GitHub and benefits from step-by-step help when publishing changes.

When helping in future sessions:

- Prefer simple explanations
- Be explicit about GitHub Desktop steps
- Keep changes practical and easy to maintain
